# LosList
a mubu/workflowy demo.

## What is LosList?

This is a demo project that mock workflowy.com & mubu.com.

## How can I run it?

* `git clone https://github.com/LosEcher/LosList.git`
* `yarn` or `npm install`
* `yarn serve` or `npm start`
